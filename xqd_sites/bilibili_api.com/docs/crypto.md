# B站搜索加密分析

## WBI 签名

### 生成流程

1. **获取密钥**: 从 `https://api.bilibili.com/x/web-interface/nav` 获取
   - `img_key`: 从 `data.wbi_img.img_url` 文件名提取（32字符）
   - `sub_key`: 从 `data.wbi_img.sub_url` 文件名提取（32字符）

2. **生成 mixin_key**:
   - 拼接: `raw = img_key + sub_key`（64字符）
   - 用固定索引表重排: `mixin_key = raw[46] + raw[47] + raw[18] + ...`
   - 取前 32 字符

3. **参数签名**:
   - 过滤参数值中的特殊字符: `!*'();:@&=+$,/?%#[]`
   - 按 key 字母排序
   - 拼接为查询字符串
   - 附加 `wts=<当前时间戳>`
   - `MD5(拼接字符串 + mixin_key)` → `w_rid`

### 索引表

```
[46, 47, 18, 2, 53, 8, 23, 32, 15, 50, 10, 31, 58, 3, 45, 35,
 27, 43, 5, 49, 33, 9, 42, 19, 29, 28, 14, 39, 12, 38, 41, 13,
 37, 48, 7, 16, 24, 55, 40, 61, 26, 17, 0, 1, 60, 51, 30, 4,
 22, 25, 54, 21, 56, 59, 6, 63, 57, 62, 11, 36, 20, 34, 44, 52]
```

### 特点

- **纯算实现**: 无需浏览器环境，纯 Python 可完成
- **无需登录**: B站搜索不强制登录
- **密钥更新**: `img_key` 和 `sub_key` 会定期更换

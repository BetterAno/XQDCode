# External Tools

- `curl_cffi`: browser-like TLS fingerprint for protocol validation.
- `webcrack`: unpack webpack/browserify and deobfuscate common patterns.
- `grpcio-tools`: compile `.proto` files when protobuf schema is available.
- `mitmproxy`: controlled HTTPS inspection when live browser evidence is insufficient.
- `websocket-client`: Python CDP bridge transport.
